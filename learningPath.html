HTML & CSS Track

Your 30-day web dev roadmap.

W1

The Basics

Days 1–7
  • What is HTML? Tags & structure
  • Headings, paragraphs, links, images
  • Lists — ordered and unordered
  • Build your first webpage
  • Intro to CSS — colors and fonts
  • The box model: margin, padding, border
  • Style your first page
W2

Layout

Days 8–14
  • CSS Flexbox — rows and columns
  • Float and clear
  • Navigation bars
  • Intro to Bootstrap
  • Bootstrap grid system
  • Bootstrap cards and buttons
  • Build a multi-section layout
W3

Interactivity

Days 15–21
  • Intro to JavaScript
  • Variables, functions, events
  • DOM — changing page content
  • Forms and input validation
  • Responsive design basics
  • CSS media queries
  • Mobile-first thinking
W4

Your Project

Days 22–30
  • Plan your personal site
  • Build Home + About pages
  • Add a contact form
  • Polish with CSS animations
  • Create a GitHub account
  • Push your project to GitHub
  • Deploy with GitHub Pages 🎉
Python Track

Your 30-day Python roadmap.

W1

Getting Started

Days 1–7
  • Install Python & VS Code
  • print(), variables, data types

Before you start

Tips from someone who's been there.

Show up every day

Even 20 minutes daily beats a 4-hour session once a week. Consistency is everything when learning to code.

Bugs are normal

Every developer googles error messages. Getting stuck doesn't mean you're bad at this — it means you're learning.

Type it, don't copy it

Typing code by hand — even from a tutorial — builds muscle memory and helps you actually understand what you're writing.

Build something real

The fastest way to learn is to have a project you actually care about. Your first site can be about anything — make it yours.
